# 📚 Overview

Personalized Learning Agent is an AI-powered educational platform designed to make learning accessible, adaptive, and engaging for everyone.

The application acts as a personal AI tutor that understands each learner's pace, learning style, and progress, then generates personalized study plans, quizzes, explanations, and learning pathways.

The platform also supports offline mode, accessibility tools, and gamification to improve the overall learning experience.

This project was developed for the **Google & Kaggle AI – Agent for Good** challenge.

---

# 🌟 Problem Statement

Millions of students struggle with:

- One-size-fits-all education
- Lack of personalized guidance
- Limited access to quality tutors
- Learning disabilities
- Poor motivation
- No adaptive roadmap

Our AI agent solves these challenges by providing an intelligent tutor that adapts to every learner.

# 🚀 Installation

Clone Repository

```bash
git clone https://github.com/yourusername/personalized-learning-agent.git
```

Open Folder

```bash
cd personalized-learning-agent
```

Install Packages

```bash
npm install
```

Run Development Server

```bash
npm run dev
```

Open

```
http://localhost:5173
```

---

# 🔑 Gemini API Setup

Create a `.env` file

```
VITE_GEMINI_API_KEY=YOUR_API_KEY
```

If no API key is provided, the application automatically switches to Offline AI Mode.
